# Third-Party Components

This project downloads and repackages third-party software during bundle build.
Those components are not covered by this repository's MIT license.

## Source Repositories

- oh-my-zsh: https://github.com/ohmyzsh/ohmyzsh
- zsh-autosuggestions: https://github.com/zsh-users/zsh-autosuggestions
- zsh-completions: https://github.com/zsh-users/zsh-completions
- zsh-syntax-highlighting: https://github.com/zsh-users/zsh-syntax-highlighting
- powerlevel10k: https://github.com/romkatv/powerlevel10k

## Ubuntu Packages

The bundle also includes Ubuntu `.deb` packages downloaded from Ubuntu mirrors.
These packages are provided under their own package-specific licenses and Ubuntu distribution terms.

## User Responsibility

If you redistribute generated offline bundles, you are responsible for:

- complying with all upstream licenses,
- preserving required notices,
- ensuring distribution terms are satisfied in your region and scenario.
